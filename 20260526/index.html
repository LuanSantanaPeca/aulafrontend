Exercício 01

Exercício 02

Exercício 03

Clique no botão para alterar a cor de fundo da tela. Deve ter 4 botões, sendo que um deles deve mostrar a cor original de fundo de tela




Exercício 04

Clique no botão para criar novo parágrafo. Você deve digitar o texto a ser inserido no parágrafo. Um novo botão deve ser criado para remover sempre o ultimo parágrafo adicionado. Quando não houver mais parágrafos, uma msg deve ser apresentada informando esta condição

Exercício 08

Exercício 09
